Graph_Gen 사용법

1. 명령 프롬프트에서 수행한다.
2. Graph_Gen을 수행하면 몇개의 graph를 생성할지 입력한다.
3. 각 graph에 대해 vertex 갯수, edge density, max edge cost를 입력한다.
4. edge density는 1.0 ~ 5.0까지 실수를 입력할 수 있다.
   값이 작으면 graph는 sparce해지며, 값이 크면 dense해진다.
5. max edge cost를 1로 주면 모든 edge cost는 1이 된다.
   이 max edge cost는 정수 값으로 1보다 크거나 같은 값을 준다.
